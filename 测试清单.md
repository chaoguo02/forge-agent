## 五、运行测试命令

```bash
# 全量测试
pytest

# 带详细输出的全量测试
pytest -v --tb=short

# 运行单个测试文件
pytest tests/test_day3.py -v

# 运行单个测试类
pytest tests/test_day3.py::TestFileReadTool -v

# 运行单个测试方法
pytest tests/test_day3.py::TestFileReadTool::test_read_existing_file -v

# 运行测试并显示覆盖率
pytest --cov=agent --cov=tools --cov=llm --cov=context --cov=entry --cov-report=term

# 跳过需要 Docker 的测试
pytest -m "not docker" -v

# 只看失败测试
pytest -q --tb=short | grep FAILED

# 最快模式（调试时使用，跳过慢测试）
pytest -x --timeout=30 -q
```
